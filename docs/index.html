Что умеет проект

Полный набор инструментов для тонкой настройки niri

Проект не заменяет niri, а делает работу с его конфигурацией понятной, безопасной и визуальной.

Мониторы

Разрешение, частота, масштаб, позиция, трансформация, VRR и отключение отдельных выходов.

Ввод

Настройка XKB, повтора клавиш, фокуса по наведению, прокрутки, тачпада, мыши и курсора.

Сочетания клавиш

Визуальная карта клавиатуры, список привязок, поиск и редактор действий niri.

Внешний вид

Тени, границы, скругления, прозрачность и эффекты размытия для окон.

Анимации

Глобальное включение анимаций, замедление, редактор easing-кривых и пресеты под сценарии.

Расположение

Промежутки между окнами, ширина колонок, центрирование, struts, фон и поведение CSD.

Рабочие пространства

Именованные workspace и привязка к мониторам для аккуратной структуры рабочего стола.

Правила окон

Условия по App ID, заголовку или namespace, размещение, размеры и визуальные эффекты.

Безопасность прежде всего

Изменения применяются аккуратно и с контролем

Перед сохранением выполняется проверка конфигурации через niri validate. Если проверка не проходит, изменения не применяются, а все правки можно откатить или восстановить из резервной копии.

  • Проверка перед сохранением
  • Работа через временные файлы
  • Резервные копии и отмена действий
  • Поддержка ручных правок через встроенный KDL-редактор
1. Редактируете Меняете параметры в привычном интерфейсе.
2. Проверяете Система запускает niri validate и блокирует сбойные правки.
3. Применяете Сохраняете изменения с возможностью отката и восстановлением из копий.

Многофайловые конфигурации

Управление сложными проектами без потери структуры

NiriMod RU умеет читать конфигурации, разделённые через include, и сохранять изменённые параметры обратно в исходные файлы. Это удобно для кастомных сборок и дополнительных слоёв окружения.

  • Поддержка файлов для ввода, мониторов, правил окон и автозапуска
  • Изменения затрагивают только известные настройки
  • Подходит для окружений с дополнительными shell-оболочками и слоями

Установка

Выберите удобный путь для своей системы

Сборка из PKGBUILD

Подходит для Arch Linux и CachyOS. Сборка через makepkg и установка пакета одним шагом.

mkdir -p ~/build
cd ~/build
git clone https://github.com/y-tretyakov/nirimod-ru.git
cd nirimod-ru
makepkg -si

Установка уже собранного .pkg.tar.zst

Если архив уже подготовлен на другой машине или в CI, достаточно установить его напрямую.

sudo pacman -U ./nirimod-ru-git-r91.20260703-1-x86_64.pkg.tar.zst

Для других дистрибутивов

Используйте установочный скрипт и выберите нужный режим через флаги.

curl -sSL https://raw.githubusercontent.com/y-tretyakov/nirimod-ru/main/install.sh | bash

Требования

Что нужно для работы

  • Linux с Wayland
  • Установленный niri
  • Python 3.12+
  • GTK4, libadwaita, PyGObject, Pycairo
  • uv для запуска из исходников

Разработка

Проверка и тестирование

uv sync --dev
uv run pytest

Для проверки PKGBUILD можно использовать makepkg и namcap.
